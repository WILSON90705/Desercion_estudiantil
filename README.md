📌 descripcion

Este proyecto es un analisis de 9500 estudiantes, en donde se
transforma y analiza datos académicos con el objetivo de detectar, las variables que mas llevan 
a los estudiantes al abandono de sus estudio. 
A través de consultas estructuradas, reportes agregados y métricas clave, 
se busca la parametrizacion de estas variables alcabo de 5 años con el fin,
de generar el reporte mas adecuado, para una institución
